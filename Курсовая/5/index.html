Центральный процессор является одним из ключевых составляющих элементов современного ПК.
Началом эпохи современных CPU можно смело назвать 1971-й год.

Отличия современных процессоров от их предков

  • Архитектура и технология производства
  • Мультипоточность и многоядерность
  • Интеграция компонентов
  • Энергоэффективность и терморегуляция

Из чего состоит процессор?

Аритметико-логическое устройство
Выполняет арифметические и логические операции
Устройство управления
Координирует работу всех компонентов процессора, управляет выполнением инструкций
Регистры
Небольшие области памяти внутри процессора для хранения промежуточных данных и адресов
Кэш-память
Быстрая память, используемая для хранения часто используемых данных и инструкций
Шина данных
Система соединений для передачи данных между процессором и другими компонентами компьютера
Модуль ввода-вывода
Отвечает за взаимодействие процессора с внешними устройствами
Механизмы управления потоком
Обеспечивают выполнение процессов и многозадачность
  • Кристалл
  • Медная крышка
  • Кристалл на плате
  • Сокет процессора
  • Схема процессора
